# 日语对话语料清洗规则文档

项目：ja-dialogue-cleaning-starter

## 1. 去噪规则

1. 删除所有颜文字、emoji、特殊符号（@#￥%&\*等）
2. 删除多余空格、换行符、制表符
3. 删除非日语字符（英文/数字除外，日常对话保留少量数字）
4. 过滤空文本、无意义单字

## 2. 敬语统一规则

- 统一转换为标准 **ですます体**（礼貌体，职场/日常通用）
- 禁止使用だ体、である体、タメ口
- 转换示例：
  だ → です
  である → です
  食べる → 食べます
  行った → 行ったです

## 3. 中式日语过滤规则

删除高频中式日语表达：
私は / あなたは / とても / すごく / 本当に / 非常に / 私の / あなたの

## 4. 不良内容过滤规则

过滤包含以下类别的文本：

- 暴力、犯罪、脅迫
- 卑猥、性的表現
- 人種・障害・性別差別

## 5. 标签化规则

情绪标签共5类：

- happy：嬉しい、楽しい、好き
- sad：悲しい、寂しい
- angry：怒る、腹立たしい
- question：疑問、どうして
- neutral：默认情绪
